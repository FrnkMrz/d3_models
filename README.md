# 3D Models

OpenSCAD models and helper files for personal 3D-printing projects.

## Files

- `*.scad`: editable OpenSCAD source files
- STL/3MF/G-code files are intentionally ignored by default, because they can become large and are usually generated artifacts

## Workflow

1. Edit the `.scad` file.
2. Render/export in OpenSCAD.
3. Slice in Bambu Studio or OrcaSlicer.
4. Commit the changed `.scad` source file.
